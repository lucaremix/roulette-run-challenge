Entrambi i giocatori scelgono Split o Steal in segreto:

• Split / Split → Entrambi ricevono 1500 Coins ciascuno
• Split / Steal → Chi ha scelto Steal ottiene 3000 Coins, l'altro 0
• Steal / Steal → Nessuno ottiene nulla

Scegli saggiamente. La fiducia può essere ricompensata... o tradita.
